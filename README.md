# Smart Task Manager Pro 

A professional Python-based task management system designed to demonstrate automated technical documentation using **Sphinx**.

##  Overview

This project simulates a real-world data pipeline structure. It manages the lifecycle of tasks, provides priority-based filtering, and includes utility helpers for data validation.

The main goal of this repository is to showcase the **Docs-as-Code** philosophy: keeping technical documentation synchronized with the source code.

##  Project Structure

* `src/` - Core Python source code (Models, Engine, Utils).
* `docs/` - Sphinx documentation source files and configurations.
* `config/` - YAML configuration files for the application.
* `data/` - Sample CSV data for demonstration purposes.

##  Requirements

- Python 3.8+
- Sphinx
- sphinx-rtd-theme (for the professional blue UI)
